```javascript
 git pull --force  <远程主机名> <远程分支名>:<本地分支名>
 // git:pull --force 强制覆盖本地的分支
```

```javascript
// 从远端拉取指定分支
git checkout -b 本地分支名称 origin/远程分支名称
// 推送到指定分支
git push origin 本地分支:远端希望创建的分支
```
